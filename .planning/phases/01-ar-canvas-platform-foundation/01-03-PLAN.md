    Create src/hooks/useSkeletonAnimation.ts:
    
    **Hook signature:**
    ```
    export function useSkeletonAnimation() {
      return { 
        rotation: [number, number, number],
        isAnimating: boolean,
        stopAnimation: () => void,
        rotationSpeed: number 
      }
    }
    ```
    
    **State:**
    - `rotation`: Current rotation angles [x, y, z], initialized as [0, 0, 0]
    - `isAnimating`: Boolean, initialized as true (auto-rotation enabled on mount)
    - `rotationSpeed`: Constant, 0.003 radians per frame (gentle rotation, adjustable)
    
    **Animation loop (via useFrame in SkeletonPreview):**
    - If isAnimating, increment rotation.y by rotationSpeed each frame
    - Wrap angles to keep them manageable (optional, but good practice)
    
    **stopAnimation function:**
    - Sets isAnimating to false (pauses rotation)
    - Called when user clicks or interacts with canvas
    
    **Implementation pattern:**
    Use Zustand store or React useState. For simplicity, use useState in a custom hook:
    ```
    const [isAnimating, setIsAnimating] = useState(true);
    const rotationRef = useRef([0, 0, 0]);
    const rotationSpeed = 0.003;
    
    const stopAnimation = useCallback(() => setIsAnimating(false), []);
    
    return { 
      rotation: rotationRef.current as [number, number, number],
      isAnimating,
      stopAnimation,
      rotationSpeed
    };
    ```
    
    Export stopAnimation and isAnimating for use by SkeletonPreview.
  </action>

  </read_first>
  <action>
    Source a skeleton model from one of these options:
    
    **Option 1: AnatomyTOOL (GitHub)**
    - Free, GPL3 license, anatomy-focused
    - Download from https://anatomytool.org/open3dmodel-create
    - Models are pre-optimized and have named mesh hierarchies (skeleton, muscles, organs)
    
    **Option 2: Sketchfab (CC-BY / CC0 filtered)**
    - Search "skeleton anatomy" filter by "Downloadable" and "Creative Commons"
    - Download in GLB format
    - Verify license is permissive (CC0 or CC-BY preferred)
    
    **Option 3: TurboSquid Free**
    - 600+ free glTF models available
    - Filter by "skeleton" or "anatomy"
    
    **Optimization steps (if model exceeds 5MB):**
    
    1. **Draco compression:**
       - Install gltf-pipeline: `npm install -g gltf-pipeline`
       - Convert: `gltfpipeline -i skeleton.glb -o skeleton-draco.glb --draco.compressionLevel=7`
       - This typically reduces file size by 50-80%
    
    2. **Decimation (reduce polygon count):**
       - Use Blender or MeshLab to reduce poly count by 50%
       - Re-export as GLB
    
    3. **Texture optimization:**
       - If model has textures, convert to KTX2 format (requires gltf-transform)
       - Or remove textures if not needed for Phase 1
    
    **Acceptance criteria for model:**
    - File size ≤ 5MB
    - Free/open license (GPL3, CC0, CC-BY)
    - Named mesh hierarchies (e.g., "Skeleton", "Muscles", "Organs" as separate objects for Phase 3 layer toggles)
    - Web-compatible format (GLB preferred, glTF+separate textures acceptable)
    - No errors when loaded via useGLTF
    
    **Where to save:**
    Place in `src/models/skeleton.glb`
    
    **Fallback if no suitable model found:**
    Create a minimal skeleton from basic Three.js primitives (boxes for bones, spheres for joints) as a placeholder. This satisfies the "test object" requirement without requiring external assets. Can be replaced with real model in Phase 2.
